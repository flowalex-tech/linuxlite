Fun Programs

Here are a few other programs I didn't think it deserved a full option in the main page, but still worth mentioning

Cool Retro Term

Lorem ipsum.

Ubuntu
sudo add-apt-repository ppa:vantuz/cool-retro-term
sudo apt update
sudo apt install cool-retro-term

Cowsay

cowsay is a program that generates ASCII pictures of a cow with a message.

Ubuntu
$ sudo apt-get install cowsay

Figlet

FIGlet is a program for making large letters out of ordinary text.

Ubuntu
$ sudo apt-get install figlet

Lolcat

Adds rainbows to your terminal.

Ubuntu
$ sudo apt-get install lolcat

Steam Locomotive

A program that shows steam locomotive on your Linux terminal for people who type the ls command backward.

Ubuntu
$ sudo apt-get install sl

Cmatrix

CMatrix is based on the screensaver from The Matrix website.

Ubuntu
$ sudo apt-get install cmatrix
